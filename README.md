# 🚀 RemotIn — Curated Remote Job Board for Indonesia

RemotIn adalah platform papan pekerjaan (job board) terkurasi yang menghubungkan talenta Indonesia dengan peluang kerja jarak jauh (remote work).

🔗 **Live Website:** [remotinjobs.com](https://remotinjobs.com)

---

## 🛠️ Tech Stack & Fitur Utama

* **Framework:** Next.js 16 (App Router), TypeScript, Tailwind CSS
* **Database & Auth:** Supabase PostgreSQL (RLS Enforced), Google OAuth, Kode OTP Manual
* **Security Hardening:** HMAC-SHA256 Session Cookie, Anti-Stored XSS (Unicode Escaping JSON-LD), Vercel Firewall Rate Limiting
* **Server Architecture:** Server-Side URL-Driven Filtering, Model Data Companies dengan Permalinks SEO (`/perusahaan/[slug]`)
* **Integrasi:** Resend API (Email SMTP), Midtrans Payment Gateway, Google Analytics 4, Cloudflare Turnstile
